## 📋 Description du projet

Cette application permet de **prédire si un client d'une entreprise de télécommunications est susceptible de résilier son abonnement** (churn).

Le modèle analyse plusieurs caractéristiques du client (ancienneté, type de contrat, montant des factures, services utilisés, etc.) pour estimer le risque de churn avec une probabilité associée.

---

## 🛠 Technologies utilisées

- **Langage** : Python
- **Interface Web** : Streamlit
- **Modèle de Machine Learning** : XGBoost
- **Prétraitement** : Scikit-learn, Pandas, NumPy
- **Sauvegarde du modèle** : Joblib
