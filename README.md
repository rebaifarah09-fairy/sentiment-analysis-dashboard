## 📋 Description du projet

Cette application permet d’**analyser les sentiments** (positif, négatif ou neutre) des commentaires clients en temps réel.  

Elle est conçue pour traiter des avis clients (comme ceux d’Amazon, de sites e-commerce, ou de réseaux sociaux) en utilisant des modèles de **NLP** pré-entraînés de **Hugging Face**.

L’interface intuitive permet à l’utilisateur de :
- Saisir un commentaire ou une review
- Analyser le sentiment instantanément
- Visualiser les résultats sous forme de jauges, graphiques et scores détaillés
- Charger plusieurs commentaires en batch

---

## 🛠 Technologies utilisées

- **Interface** : Streamlit
- **Modèles NLP** : Hugging Face Transformers
- **Analyse de sentiments** : Modèles pré-entraînés (ex: DistilBERT, RoBERTa, etc.)
- **Visualisation** : Plotly
- **Traitement de données** : Pandas
