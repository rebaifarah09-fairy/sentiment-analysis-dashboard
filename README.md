## 📋 Description du projet

Ce projet permet de **prédire si un client d'une entreprise de télécommunications est susceptible de résilier son abonnement** (churn).

Il inclut :
- Nettoyage et prétraitement des données
- Feature Engineering
- Entraînement d’un modèle **XGBoost**
- Explicabilité avec SHAP (optionnel)
- Interface web interactive avec **Streamlit**

## ✨ Fonctionnalités

- Prédiction en temps réel via formulaire
- Visualisation des features importantes
- Interface moderne et intuitive
- Modèle performant avec bonnes métriques (Accuracy + ROC AUC)

## 🛠️ Technologies utilisées

- **Python**
- **Pandas & NumPy**
- **Scikit-learn**
- **XGBoost**
- **Streamlit** (Dashboard)
- **Joblib** (Sauvegarde du modèle)
